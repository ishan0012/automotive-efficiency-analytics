# 🏎️ Automotive Intelligence & Fuel Efficiency Predictor

## 📌 Project Overview
This project is a comprehensive data science study of the **Automotive Industry**. It combines **Exploratory Data Analysis (EDA)** with **Machine Learning** to understand and predict fuel efficiency (MPG). Using a dataset of 390+ vehicle models, the system identifies how engineering specs like horsepower and weight impact environmental performance across USA, Europe, and Japan.

## 🛠️ Tech Stack
- **Analysis:** Pandas, NumPy
- **Visualizations:** Seaborn, Plotly (Interactive), Matplotlib
- **Machine Learning:** Scikit-Learn (Linear Regression)

## 🚀 Key Features
- **Interactive EDA:** Dynamic scatter plots to visualize the trade-off between power and efficiency.
- **Regional Benchmarking:** Comparative analysis of manufacturing standards between global regions.
- **Predictive Engine:** A regression model capable of predicting a vehicle's MPG based on its mechanical configuration with ~70-80% accuracy.
- **Data Pipeline:** Automated cleaning of engineering datasets, handling missing values in horsepower records.

## 📊 Results
The analysis confirmed a strong negative correlation between vehicle weight and fuel economy. The regression model provides a baseline for estimating fuel consumption for new vehicle prototypes.
